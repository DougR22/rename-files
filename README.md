# File Rename Utility

A two-program workflow for batch renaming PDF files using a mapping system.

## Programs

### 1. extract.py

**Purpose**: Extract PDF filenames from the current directory and save them to a file.

**Functionality**:
- Lists all `.pdf` files in the script's directory
- Saves each filename to `extract_filenames.txt` (one per line)
- Prints extracted filenames to console
- Displays total count of processed files

**Usage**:
```bash
python extract.py
```

**Output**:
- `extract_filenames.txt` - Contains the original PDF filenames

**Example**:
```
Extracted: document1.pdf
Extracted: document2.pdf

Total files processed: 2
Results saved to: extract_filenames.txt
```

---

### 2. replace.py

**Purpose**: Rename files using a 1:1 mapping system with validation and logging.

**Functionality**:
- Reads original filenames from `extract_filenames.txt`
- Reads replacement names from `replace_filenames.txt`
- Validates new filenames for invalid Windows characters: `< > : " / \ | ? *`
- Performs dry-run by default (no actual renames)
- Optionally applies the renames with `--apply` flag
- Logs all operations to `rename_log.txt`
- Logs validation errors to `validation_log.txt`

**Validation**:
- Checks each new filename character by character to ensure there are no characters that are disallowed by MS Windows standards
- Reports invalid characters with their position in the filename
- Prevents rename operations if validation errors exist
- Saves detailed validation report to `validation_log.txt`

**Usage**:

Dry-run (default - shows what would happen):
```bash
python replace.py
```

Apply renames:
```bash
python replace.py --apply
```

**Input Files Required**:
- `extract_filenames.txt` - Original filenames (generated by extract.py)
- `replace_filenames.txt` - Desired new filenames (create manually)

**Output Files**:
- `rename_log.txt` - Log of all rename operations (DRY_RUN, RENAMED, ERROR, CONFLICT, SKIP)
- `validation_log.txt` - Validation results with any invalid characters found


<br>

## Example Workflow

1. Run extract.py to get list of PDFs:
```bash
python extract.py

EXAMPLE OUTPUT:

angular6forenterprisereadywebapplications.pdf
cleancodeinjavascript.pdf
fullstackwebdevelopmentwithtypeScript5.pdf
```

2. Edit `replace_filenames.txt` with new names (one per line, matching to extract_filenames.txt, in same order), for example:
```
Angular - Angular 6 for Enterprise-Ready Web Applications.pdf
General - Clean Code in JavaScript.pdf
TypeScript - Full-Stack Web Development with TypeScript 5.pdf
```

3. Validate and preview changes with dry-run:
```bash
python replace.py
```
4. Check dry-run results in `rename_log.txt` and `validation_log.txt`

5. If validation passes and preview looks good, apply renames:
```bash
python replace.py --apply
```

6. Check final results in `rename_log.txt` and `validation_log.txt`

<br>

## Error Handling

- **Validation Errors**: Invalid Windows fielname characters prevent execution. Fix `replace_filenames.txt` and re-run.
- **Missing Files**: Only processes files that exist with matching old names.
- **Conflicts**: Skips renames where destination filename already exists.
- **Rename Errors**: Prints error messages in ALL CAPS to console and log files.

## Tips

- Always run with dry-run first to verify changes
- Ensure `replace_filenames.txt` has same number of lines as `extract_filenames.txt`
- Invalid Windows filename characters will prevent the script from proceeding
- Check both log files after running to verify results
